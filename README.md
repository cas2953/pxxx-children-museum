# 國家兒童未來館｜設計團隊情報板

提案前決策用的互動式情報板：彙整國際標竿展館、各國代表性設計團隊、與本案盤點的候選廠商，
含代表性說明、關鍵人物、強項關鍵字、官網／聯絡／作品集、全球分佈地圖，以及可勾選比較的「我的清單」（含與台北時差）。

## 線上瀏覽
本專案是單一檔案的靜態網頁，開啟 `index.html` 即可使用（可離線，世界地圖已內嵌）。
建議以 GitHub Pages 發佈成網站（見下方）。

## 內容
- `index.html` — 互動情報板（標竿展館 5｜標竿展館設計團隊 6｜國際代表團隊 19｜候選廠商 10）

## 功能
- 搜尋、依類別（候選廠商／展館設計團隊／國際代表團隊／標竿展館）與地區篩選
- 每筆含：代表性說明、關鍵人物／要角、強項關鍵字、代表作、相關性、官網／作品集／聯絡
- 全球分佈地圖：四類用不同圖標（▲展館 ◆展館設計團隊 ●候選廠商 ■國際團隊），hover 顯示該國團隊、click 篩選
- 「我的清單」：勾選加入 → 獨立比較頁，顯示完整資訊＋地點＋與台北即時時差，可列印／存 PDF

## 以 GitHub Pages 發佈成網站
1. 將本資料夾的檔案推送到一個 GitHub 儲存庫（Public）。
2. 進入 repo 的 **Settings → Pages**。
3. **Source** 選「Deploy from a branch」，分支選 `main`、資料夾選 `/ (root)`，按 Save。
4. 約 1 分鐘後即可在 `https://<你的帳號>.github.io/<儲存庫名稱>/` 開啟。
